# Kenko

Mobile App to create and track workouts

[Demo](http://budget-tracker.info/)

## Getting Started

### Installing

npm:

```
npm install
npm start
```

yarn:

```
yarn install
yarn start
```

## Built With

- [React Native](https://facebook.github.io/react-native/)
- [Styled Components](https://www.styled-components.com/)
- [Expo](https://expo.io/)
